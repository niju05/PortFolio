Smart Switch Board

Smart Switch Board is an IoT-based system that converts traditional electrical switchboards into smart, remotely controllable devices using mobile applications and wireless communication.

Project Overview

This project enables users to control home appliances such as lights and fans remotely. It integrates hardware with a mobile application, allowing users to turn devices ON/OFF, schedule operations, and monitor usage efficiently.

Technologies Used

  • Flutter (Mobile App Development)
  • Firebase Realtime Database
  • Embedded C (ESP32 Programming)
  • Wi-Fi Communication

Hardware Components

  • ESP32 Microcontroller
  • Relay Module
  • Switchboard Integration Circuit
  • Power Supply Unit

Flutter Mobile App

A Flutter application was developed to provide a user-friendly interface for controlling devices. The app allows real-time switching, scheduling, and synchronization with the ESP32 using Firebase.

Key Features

  • Remote ON/OFF control of appliances
  • Real-time updates using Firebase
  • Scheduling system for automation
  • Low-cost and easy installation
  • Works with existing switchboards
