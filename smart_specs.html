Smart Specs

Smart Specs is an AI-powered assistive system designed for visually impaired individuals. It helps users navigate their surroundings using real-time voice feedback and intelligent object detection.

Project Overview

The system captures real-world visuals using a camera and processes them using AI models to detect obstacles and directions. Based on the output, it provides voice instructions such as "Move Left", "Move Right", or "Stop", ensuring safe navigation.

Technologies Used

  • Python (AI Model Integration)
  • Machine Learning (Object Detection)
  • Firebase (Real-time data handling)
  • Flutter (Mobile App Development)

Hardware Components

  • ESP32 (Microcontroller)
  • Camera Module
  • Ultrasonic Sensors
  • Battery Module
  • Bluetooth / WiFi Module

Flutter Mobile App

A Flutter-based mobile application was developed to connect with the device, display system status, and allow user interaction. The app ensures seamless communication between hardware and software using Firebase and real-time updates.

Key Features

  • Real-time navigation assistance
  • Voice-based alerts and commands
  • Obstacle detection using AI
  • Lightweight and wearable design
  • Mobile app integration
